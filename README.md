# orage-todo

orage-todo is a shell script to display upcoming envents from orage calendar.

* support (  ~~HOURLY~~  ,  DAILY, WEEKLY, MONTHLY, YEARLY) periodic events

## Require

* [orage](https://github.com/xfce-mirror/orage)

## Useage

    chmod +x orage-todo
    ./orage-todo
